# QCAPI

**qcapi** is a set of APIs for querying QC reference information and activate a process `watchwsi` to monitor specified WSI folders in scanner workstation, move WSI files to preset watch folder for automatocally running AIxMED model inference WSI files if scanned whole slide image created.  
**qcapi** will parse model analyzed metadata and compile QC refernece data to respond the request for querying QC reference information.
